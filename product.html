The Product

Big Knit works by combining firmware, electronics, and mechanical subsystems. With each piece working correctly in conjunction with the others, we get a highly-autonomous knitting machine. All the user needs to do is cast the yarn onto the needles, tell the machine how long and how wide the knit should be, and then cast off once the machine is done.

[INSERT MORE PRECISE/BETTER WORDED EXPLANATION OF THE SYSTEM] To make this happen, we have created a mounted gantry with a pulley system. The pulley is driven by a stepper motor mounted onto one end of the gantry. This linear motion drives a cam that makes a wave as it slides across knitting needles. All of this movement is dictated by code coming from an Arduino Uno and hooked up to a solderless breadboard to control limit switches, drivers, and other electronics.
